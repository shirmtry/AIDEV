====================================================
PROJECT GOAL
============

Build a complete AI-powered agricultural assistant for coffee farmers in the Central Highlands of Vietnam.

The system must:

1. Accept plant images.
2. Detect plant diseases using the provided HuggingFace model.
3. Map model predictions to diseases.json.
4. Retrieve weather data from OpenWeatherMap.
5. Calculate disease risk scores.
6. Generate recommendations using CrewAI/Groq.
7. Display everything in Streamlit.
